Step 4: Generate symbolic links

Back to Organizing experimental data

After checking that the description files generated in the previous step look OK we are ready to generate the symbolic links using the command:

somsds_update_rec bcgs

You can run the command above yourself to see the output that it produces. You should pay attention for warnings that may indicate that something went wrong during the generation of the tag descriptor files.

What now?

Congratulations! Your data is now properly organized. All the important files, that you are likely to use in your analyses are now easily available from the raw directories, as discussed in The SOMSDS data structure. You can now start analyzing your data.
